Projetos

Nesta seção apresento projetos desenvolvidos com foco em análise de dados, visualização e desenvolvimento front-end. Cada projeto detalha o problema, abordagem técnica, ferramentas utilizadas e principais resultados, e apresenta algumas desenvolturas e conhecimentos tecnológicos.

01

Relatório de Vendas - Data Analysis

Excel • Power BI • Analise de Dados • Vendas

Este projeto consiste na análise estratégica de um conjunto de dados de vendas, com o objetivo de identificar padrões de comportamento, oportunidades de crescimento e riscos operacionais no desempenho comercial. A partir de um dashboard público, cuja criação não é de minha autoria, concentrei minha atuação na análise e interpretação dos dados, conduzindo uma leitura orientada por indicadores-chave, como faturamento, recorrência, canais de aquisição, performance por produto e eficiência da equipe de vendas. O trabalho resultou na consolidação de insights acionáveis e direcionamentos estratégicos, evidenciando a capacidade de transformar dados em decisões, com foco em retenção, expansão de receita e otimização da operação.

03

Chatbot com IA usando n8n e Groq e registro em nuvem

N8N • Groq • API • LLM • Workflow • Google Sheets • Nuvem

Desenvolvimento de um chatbot automatizado utilizando n8n para orquestração de workflows e integração com modelos de linguagem via API da Groq. O sistema recebe mensagens dos usuários, envia as solicitações para processamento por IA e retorna respostas automaticamente. Todas as interações são registradas em Google Sheets, permitindo armazenamento em nuvem e histórico completo das conversas para análise e monitoramento.

05

Lume App – Aplicação de Recomendação de Filmes com IA

.NET MAUI • C# • API • UI/UX • Consumo de Dados • Mobile • Xaml • Kanban • Figma

Aplicação mobile desenvolvida com .NET MAUI focada em recomendação personalizada de filmes e conteúdos culturais. O sistema coleta preferências do usuário por meio de perguntas iniciais e constrói um perfil persistente, permitindo sugestões cada vez mais refinadas ao longo do uso.

O projeto envolve integração com APIs externas, manipulação de dados, construção de interface responsiva e lógica de personalização baseada em comportamento do usuário, demonstrando aplicação prática de conceitos de experiência do usuário (UX) e inteligência de recomendação.
