# Portfólio - Sandokam Bussola

Este repositório contém o site de portfólio profissional de Sandokam Bussola, apresentando experiências, formação e projetos pessoais.

💻 **Acesse o site online:** [Portfólio no GitHub Pages](https://seu-usuario.github.io/nome-do-repositorio)

## Tecnologias

- HTML
- CSS
- JavaScript (interações dinâmicas no currículo)

## Estrutura do Projeto

- `index.html` — página inicial com introdução
- `curriculo.html` — experiência, formação e menu interativo
- `portfolio.html` — projetos realizados
- `contato.html` — formulário de contato
- `css/style.css` — estilos globais
- `arquivos/` — imagens e currículo

## Como Executar

Basta abrir o arquivo `index.html` em qualquer navegador moderno ou acessar o link online acima.

## Deploy

O site está publicado no GitHub Pages, mas também pode ser hospedado em outros serviços estáticos como Netlify ou Vercel.

## Observações

Este portfólio é **responsivo**, focado em usabilidade e demonstra habilidades em front-end.
